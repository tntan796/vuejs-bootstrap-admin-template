## Installation of Excello Template 
### Single Line Installation ( Yarn Install )
```
# install dependencies
yarnpkg install
or,
yarn install

# serve with hot reload at localhost:8080
yarn run dev

# build for production
yarn run build
```
Or,  
### NPM Installation
```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production
npm run build
```
